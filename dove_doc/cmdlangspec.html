Intro

The Commandline is one of the most central features of this Toolcollection. All operations including editing the configuration can aswell be executed via the commandline. This section of the documentation describes the syntax of Commandline commands.

Scripts

The CommandLine can also execute scripts (run -s scriptfile)

Values types

The commandline provides several datatypes. By default the datatype is specified implicitly by the syntax. Nevertheless there are several operations that require specifying the datatype aswell as some special cases.

  • Text

    Text can be created in this way: "some text"

  • Integer

    The syntax for integral numbers is: 12415

  • Double

    The syntax for real numbers is: 1241.234

  • Boolean

    Truthvalues have several representations: Either, if specified in numbers, 0 is false and 1 true, or directly via true and false.

There also exist specific operations for each datatypes. These will be discussed later on in separate paragraphs.

Variables

This paragraph describes the creation and usage of variables. The variable structure in the commandline provides the following options: variables can be made typesafe. And they can be secured to not be overwritten by newer values. Creating a new variable:
set a="hello world"
This code generates a new variable a with the text "hello world" as value.
